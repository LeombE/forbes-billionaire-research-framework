# Source Quality and Legal Compliance Skill

Use this skill whenever collecting or citing sources.

## Rules
- Respect robots.txt, terms, paywalls, login walls, and rate limits.
- Do not bypass access controls.
- Prefer official and primary sources.
- Use reputable secondary sources only for background or when primary sources are unavailable.
- Track source URL, publisher, title, author if available, date, accessed_at, fields supported, and evidence note.
- For images, track rights/license status, caption, alt text, and reason for use.
- Unsupported facts must be removed or labeled as unknown.

## Citation table fields
source_id, person_name, field_name, claim_supported, source_title, source_url, publisher, author, publication_date, accessed_at, evidence_note, reliability_tier, limitations
