# Image, Chart, and Visual Evidence Policy

Use visuals only when they add business information.

## Preferred visuals
- Wealth history line charts generated from project data.
- Industry/country/wealth-engine distribution charts.
- Ownership maps created from cited data.
- Segment revenue/profit charts from company filings.
- Market cap, stock price, or valuation bridge charts from sourced data.
- Timeline diagrams showing major inflection points.
- Value-chain diagrams for the person’s main industry.

## External images
External images are allowed only when they are informative and permitted. For each image, record:
- image_id
- person_id/name
- source_url
- publisher/owner
- title/caption
- license/usage status
- accessed_at
- local_path
- alt_text
- why_this_image_adds_information

Do not use decorative portraits unless the user explicitly asks for a magazine-style report and rights are clear.

## AI-generated/conceptual diagrams
Generated diagrams can be used for explanation, not evidence. Label them clearly as “Conceptual diagram generated from sourced analysis.” Do not imply the diagram itself is a source.

## Report rules
Every figure must have:
- Figure number
- Caption
- Source note
- Alt text
- Link to underlying data or citation row
